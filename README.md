# 📊 Dashboard Inteligente de Estoque com IA

## 🚀 Funcionalidades
- Upload de CSV
- KPIs automáticos
- Dashboard interativo
- Insights com IA (Gemini)

## 🛠 Tecnologias
- Python
- Streamlit
- Pandas
- Gemini API

## ▶️ Como rodar
1. Criar ambiente virtual
2. Instalar dependências
3. Configurar .env
4. Executar:
python -m streamlit run app/main.py
